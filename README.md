# British Airways Customer Booking Prediction

##  Project Overview
This project is part of the British Airways Data Science simulation on Forage. The goal was to predict whether a customer would complete a booking based on their behavior and preferences.

## Key Results
- **Model:** Random Forest Classifier
- **Accuracy:** 84.9%
- **Top Drivers:** Purchase Lead, Flight Hour, and Length of Stay.

##  Tools Used
- Python (Pandas, Scikit-Learn, Matplotlib)
- Jupyter Notebook
- Random Forest Algorithm

## Content
- "Getting Started.ipynb": Full data analysis and modeling code.
- "customer_booking.pptx": Final summary for management.
